# Getting Help

Resources and contacts for repository standards and guidelines.

## Documentation

- Review the [README](README.md) for an overview of the standards
- Check the [Quick Start Checklist](Quick-Start-Checklist.md) for new repositories
- Review [Current Status](Current-Status.md) to see what's been implemented
- Browse the [Reference Guide](README.md#reference-guide) for detailed standards
- Follow [Workflow Guides](Workflow-Guides.md) for step-by-step processes
- Use [Quick Reference](Quick-Reference.md) for quick checklists
- See [Templates](Templates.md) for file templates and examples

## Contribution Guidelines

- Check `CONTRIBUTING.md` in your repository for contribution guidelines
- Follow the standards outlined in this alignment document
- Ensure all P0 requirements are met before submitting PRs

## Support

- Contact maintainers listed in `CODEOWNERS` file
- Report security issues per `SECURITY.md` in your repository
- Review section-specific documentation for detailed guidance
